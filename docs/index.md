# Filing-Date-Clean Ownership Network Fragility

This site documents a point-in-time asset-pricing pipeline built from institutional ownership disclosures and dynamic common-ownership networks.

The final robustness pack's best 1-month 25 bps net portfolio is `ridge_all` with annualized net return **7.88%**, Sharpe **0.795**, and Newey-West t-stat **2.65**.
